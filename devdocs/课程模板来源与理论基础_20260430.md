## 1. 来源诚实声明

模板本身由开发过程中系统化综合生成，并非从某篇特定论文中直接摘录。但其背后的教学设计逻辑植根于广泛认可的教育学经典研究框架和前沿成果，这些知识在 LLM 训练数据中以压缩形式存在。模板 = 对这些框架的工程化转译——将其转化为可被 AI 执行的 6 模块结构化约束。

## 3. 关键框架简要说明

### 3.1 Bloom 认知分层 (1956, 修订 2001)
将学习目标从低阶到高阶排列：记忆 → 理解 → 应用 → 分析 → 评价 → 创造。模板的三阶段（foundation → practice → assessment）是对该框架的简化工程映射——基础阶段对应记忆/理解，工具实践对应应用/分析，综合应用对应评价/创造。

### 3.2 Merrill 首要教学原则 (2002)
五项原则：**激活**已有知识 → **展示**新知识 → **应用**新知识 → **整合**到真实世界。模板的 `scene_hook` 对应激活，`full_content` 对应展示，`code_example`+`skim_summary` 对应应用，`prereq_adaptive` 对应整合。

### 3.3 Sweller 认知负荷理论 (1988)
三种认知负荷：内在负荷（内容本身复杂度）、外在负荷（呈现方式带来的负担）、相关负荷（有意义的学习投入）。速成精简模板的核心设计目标就是最小化外在负荷——去掉铺垫、用列表、结论先行——让学员将有限的工作记忆全部投入到内在和相关负荷上。

### 3.4 Refutation Text 效应 (Guzzetti, 1993; Tippett, 2010)
大量实验证据表明：以"有人误认为 X → 实际上 Y"结构呈现的信息，比单纯陈述正确信息更能有效地纠正误解。模板中的 `misconception_block` 模块直接采用了这一范式（"⚠ 很多人误认为…实际上…"）。

### 3.5 Cognitive Apprenticeship (Collins, Brown & Newman, 1987)
传统学徒制的认知化版本：建模 → 辅导 → 脚手架 → 表达 → 反思 → 探索。实操导向模板的"分步编号操作步骤 + 验证方法"对应了建模-辅导-脚手架链条。

### 3.6 Vygotsky 最近发展区 (1978)
学习最有效的区域在"学习者能独立完成"和"在帮助下能完成"之间。模板的 `prereq_adaptive` 模块为不同基础的学员提供差异化扩展，正是这一原则的体现。

### 3.7 Conceptual Change Theory (Posner et al., 1982)
学习者持有先入为主的错误概念（misconceptions 或 alternative conceptions），教学必须让这些概念变得"不满意的、不可理解的、不可信的"，然后用正确概念替代。理论基础模板强调概念辨析和误解纠正，根源在此。

### 3.8 Threshold Concepts (Meyer & Land, 2003)
某些概念是学科的"门户"——一旦掌握，学习者看待整个领域的方式会发生质变（不可逆、整合性的转变）。理论基础模板的"核心概念定义 → 底层原理 → 与其他概念的关联"结构设计就是为了帮助学生跨越这些阈值。

## 4. 比赛答辩建议措辞

如果被问到模板的来源或理论基础，可以如实表述：

> "模板的设计逻辑借鉴了教学设计领域的成熟研究框架——包括布鲁姆认知分层理论、斯威勒的认知负荷理论、梅里尔的首要教学原则，以及反驳文本在科学教育中的研究证据。
>
> 在此基础上，我们将这些框架转化为可被大语言模型执行的 6 模块结构约束：
> 场景导入（锚定教学）→ 速览要点（先行组织者）→ 正文（展示+应用）→ 代码示例（应用+整合）→ 误区纠正（反驳文本）→ 自适应扩展（最近发展区）。
>
> 模板本身是我们的工程化产物，但其背后的每一个模块设计都有充分的学术文献支撑。我们不是在凭空设计——我们是在将经典教学设计原则编码为 AI 可执行的 Prompt 约束。"

## 6. 6 模块结构与理论的对应关系一览

```
scene_hook         → Anchored Instruction (CTGV, 1990)
                      + Situated Learning (Lave & Wenger, 1991)

skim_summary        → Advance Organizer (Ausubel, 1968)
                      + Cognitive Load Theory — signaling (Sweller, 1988)

full_content        → Merrill's First Principles (2002) — demonstration + application
                      + Bloom's Taxonomy — progressive complexity

code_example        → Cognitive Apprenticeship — modeling (Collins et al., 1987)
                      + Worked Examples Effect (Sweller, 2006)

misconception_block → Refutation Text (Guzzetti et al., 1993; Tippett, 2010)
                      + Conceptual Change Theory (Posner et al., 1982)

prereq_adaptive     → Zone of Proximal Development (Vygotsky, 1978)
                      + Differentiated Instruction (Tomlinson, 1999)
```
